# Changelog

## 1.0.2 (2026-05-22)

### 修复
- 页面缩放后链接点击错位、空白处误触远处链接（改用 `anchorAt` 精确命中）
- 索引列表重复别名（`Class` / `类` / 模块路径等）合并为 `类名::类名`

### 开发
- 全功能 UI 自动化测试 `ui_full_test`（不含打印）

## 1.0.1 (2026-05-22)

### 新增
- 鼠标侧键（X1/X2）前进、后退，应用内任意区域可用
- 浏览历史记录同页锚点跳转（后退可逐步回到先前段落）

### 改进
- 预构建关键词与全文索引，启动即可用
- 页面缩放更快，保留滚动位置
- 函数声明随缩放调整；访问函数表保持原文样式（非链接不着色为绿色）
- 后退/前进按钮状态与历史同步

### 修复
- 同页多次跳转后后退一次跳过多步的问题
- 点击空白取消选中、减少误滚动

## 1.0.0 (2026-05-21)

- 首个公开发布：Qt 6 简体中文 API 离线阅读、多分屏、文档管理、索引与全文搜索
